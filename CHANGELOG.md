# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## 0.1.0

### Added

- Initial release of libsignal Dart/Flutter FFI bindings
- Cross-platform support: Android, iOS, Linux, macOS, Windows
- Build hooks for automatic native library download
- GitHub Actions CI/CD for building native libraries
- Basic Dart API structure with exception handling
- Rust/Cargo build system for native compilation
- cbindgen integration for C header generation

### Infrastructure

- Makefile for common development tasks
- FVM (Flutter Version Management) integration
- Comprehensive GitHub Workflows for build and publish
